# Portfolio Webpage

A modern, responsive portfolio website showcasing web development skills and projects.

## Overview

This is a single-page portfolio website built with HTML, CSS, and JavaScript. It features a clean, professional design with smooth animations and interactive elements.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI/UX**: Professional color scheme with smooth hover effects and transitions
- **Interactive Navigation**: Smooth scrolling to different sections
- **Dynamic Recommendations**: Visitors can add recommendations via a form
- **Skills Showcase**: Display of technical skills with logos
- **Project Portfolio**: Showcase of 3 featured projects with images and descriptions

## Sections

1. **About Me**: Personal introduction with profile picture and bio
2. **Skills**: Display of technical skills (HTML5, CSS3, JavaScript, Git, React, Node.js)
3. **Projects**: Featured projects including:
   - Responsive E-Commerce Website
   - Weather Dashboard App
   - Task Management Tool
4. **Recommendations**: Client testimonials with the ability to add new ones

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Vercel Analytics

## Color Scheme

- Background: `#F4F4F4` (Light Gray)
- Header/Navigation: `#2E2E2E` (Charcoal)
- Accent: `#4A90E2` (Soft Blue)
- Text: `#333333` (Dark Gray)
- Cards: `#FFFFFF` (White) with light shadows

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/lincolncommercialsolutions/Portfolio-Webpage.git
```

2. Navigate to the project directory:
```bash
cd Portfolio-Webpage
```

3. Open with a local server:
```bash
python3 -m http.server 8000
```

4. Visit `http://localhost:8000` in your browser

## Deployment

This site is deployed on Vercel for fast, reliable hosting with built-in analytics.
